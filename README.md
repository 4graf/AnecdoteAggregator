> ***Разработка велась, соблюдая принципы DDD.***

- [x] Архитектура DDD
- [x] БД и миграции
- [x] REST API
- [x] Сервис аутентификации (JWT токены)
- [ ] Docker
- [ ] Юнит тестирование
- [ ] API тестирование
- [ ] Сервис для парсинга анекдотов с паблика вк
- [ ] Сервис с телеграм ботом
